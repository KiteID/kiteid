# KiteID Brand Kit

Official brand assets and design system for **KiteID** — the identity layer for the agentic economy on Kite blockchain.

## What's inside

```
kiteid-brand/
├── BRAND_GUIDELINES.md       Full brand guide (read this first)
├── colors/
│   ├── palette.json          Design tokens (machine-readable)
│   ├── tokens.css            CSS custom properties
│   └── tailwind.preset.js    Tailwind CSS preset
├── typography/
│   └── type-scale.md         Font system, sizes, weights
├── logo/
│   ├── kiteid-wordmark.svg              Default logo (light bg)
│   ├── kiteid-wordmark-mono-dark.svg    Single color (dark on light)
│   ├── kiteid-wordmark-mono-light.svg   Single color (light on dark)
│   ├── kiteid-wordmark-premium.svg      Bronze gradient variant
│   ├── kiteid-favicon.svg               Gold orb favicon
│   └── kiteid-icon-mark.svg             Standalone icon
└── usage/
    ├── do-and-dont.md        Logo usage rules
    └── clearspace.md         Padding & sizing rules
```

## Quick reference

**Primary colors**
- Cream Light `#FAF7F0` — page background
- Carbon `#141414` — primary text
- Burnished Gold `#C9986A` — signature accent / CTA
- Bronze Core `#9B8564` — Kite ecosystem heritage color

**Typography**
- DM Sans (Kite ecosystem standard)
- JetBrains Mono (addresses, code)

**Theme**
- Warm parchment light theme (default)
- Optional dark mode in V2

## Philosophy

KiteID inherits Kite blockchain's parchment/bronze visual heritage and extends it with a signature burnished gold accent. The aesthetic is **warm, premium, and antique-modern** — evoking trust, identity, and the timeless quality of inscribed names.

No neon. No sterile gray. No generic AI gradients.
