## What is KiteID?

KiteID is an ENS-style naming service on the **Kite AI** network. It maps human-readable `.kite` names to on-chain addresses, smart contracts, and AI agent identities.

Instead of sharing `0xBD6a09D7227F56E79327d680183317C10A1370Df`, you share `alice.kite`.

## Why Kite AI?

Kite AI is an Avalanche L1 subnet built for AI-native applications:

| Property       | Value                             |
| -------------- | --------------------------------- |
| Chain ID       | 2366 (mainnet), 2368 (testnet)    |
| Block time     | ~1 second                         |
| Gas cost       | Sub-$0.000001                     |
| Consensus      | PoAI (Proof-of-AI)                |
| Native token   | KITE                              |
| Architecture   | EVM-compatible Avalanche L1       |

## What can you do with a `.kite` name?

- **Point it to your wallet** — others send KITE to `yourname.kite`
- **Set a primary reverse record** — your address resolves back to `yourname.kite`
- **Attach arbitrary records** — website URL, avatar, social handles, IPFS hashes
- **Use it as an agent identity** — AI agents carry persistent `.kite` identities (V2)
