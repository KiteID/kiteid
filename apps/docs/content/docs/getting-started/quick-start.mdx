## Before you start

You need:
- A wallet (MetaMask, Coinbase Wallet, or any WalletConnect-compatible wallet)
- Testnet KITE tokens — get them from the [faucet](https://faucet.gokite.ai) (0.5 KITE per drip)
- At least **5 KITE** for a 5+ character name (one year)

<Callout title="Testnet only">
  KiteID is currently deployed on the Kite AI **testnet** (Chain ID 2368). Mainnet launches when Kite AI mainnet opens.
</Callout>

## Step 1 — Search

Go to [kiteid.xyz](https://kiteid.xyz) and type the name you want.

- If it shows **Available** → proceed to register
- If it shows **Taken** → try a different name or view the owner's record

## Step 2 — Commit

Registration uses a **commit-reveal protocol** to prevent front-running:

1. Click **Commit registration** — this submits a sealed hash on-chain
2. Wait **60 seconds** — this is the minimum age required by the protocol
3. Your commitment expires after **24 hours** — complete the reveal within that window

## Step 3 — Register (reveal)

After the 60-second wait, click **Register name** to finalize. This mints your name as an on-chain record and optionally sets it as your primary reverse record.

## What's next?

- Set records (website URL, avatar, social handles) on your name's detail page
- Add `.kite` to your wallet's address book
- [Read about pricing](/docs/concepts/pricing) to understand renewal costs
