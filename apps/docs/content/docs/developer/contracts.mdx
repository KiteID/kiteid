## Contract architecture

```
KiteController (UUPS proxy)
  └── manages registrations + renewals
  └── calls KiteBaseRegistrar to mint/burn
  └── calls KiteResolver to set initial records

KiteBaseRegistrar (ERC-721)
  └── mints name NFTs
  └── tracks expiry timestamps

KiteRegistry
  └── maps namehash node → (owner, resolver, TTL)

KiteResolver (ERC-137 + ERC-181)
  └── stores addr, text, contenthash per node

KiteReverseRegistrar
  └── manages addr.reverse claims

LinearPremiumPriceOracle
  └── computes base + Dutch-auction premium
```
